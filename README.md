
# Channel Profiler

## Install

`go get github.com/kuche1/channelprofiler`

Or if you want a specific tag:
`go get github.com/kuche1/channelprofiler@v0.0.1`

Or if you want a specific commit:
`go get github.com/kuche1/channelprofiler@7fab08e923c1a03b64b2bb90eac3463ee7658779`

Or use a shortened commit:
`go get github.com/kuche1/channelprofiler@7fab08e`
